About This Project

Photo Stories – Every Image Has a Story is a simple and beginner-friendly web application created as a Mini Project for the 3rd Semester of B.Tech Computer Science & Engineering. This project allows users to upload photos, write short stories, and save them locally using the browser's localStorage. It demonstrates basic skills in HTML, CSS, and JavaScript.

About Us

We are a team of CSE students who worked together to build this project. Our aim was to practice core web development skills and create something useful and easy to use.

Project Purpose

The main purpose of this project is to learn and practise front-end development by building an interactive web app. It helped us learn image handling, DOM manipulation, localStorage usage and collaborative development.
